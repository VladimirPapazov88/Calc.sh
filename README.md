Classic calculator for LINUX sh, bash or zsh

Installation:
  1. run in terminal "git clone https://github.com/VladimirPapazov88/Calc.sh.git"
  2. cd "Calc.sh"
  3. chmod +x calc.sh
  4. ./calc.sh
  
Have a happy using file!!!!
